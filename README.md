# 2026 GNS App

## 프로젝트 개요

2026 GNS App 프로젝트입니다.

## 시작하기

### 필수 요구사항

- Node.js (버전 18 이상 권장)
- npm 또는 yarn

### 설치

```bash
npm install
# 또는
yarn install
```

### 실행

```bash
npm run dev
# 또는
yarn dev
```

## Git 저장소 연결

Git이 설치되어 있다면 다음 명령어로 저장소를 초기화하고 원격 저장소와 연결할 수 있습니다:

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/keywordbridge-source/2026_gns_app.git
git branch -M main
git push -u origin main
```
