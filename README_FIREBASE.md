# 🔥 Firebase 연결하기 - 초보자용

## 📌 간단 요약
Firebase에서 다운로드한 파일을 앱 폴더에 넣으면 됩니다!

---

## 🎯 3단계로 끝내기

### ✅ 1단계: 파일 다운로드

1. **웹 브라우저 열기** (Chrome, Edge 등)
2. **주소 입력**: `console.firebase.google.com`
3. **프로젝트 선택**: `gns-app-45d6d`
4. **설정 아이콘 클릭** (왼쪽 아래 ⚙️)
5. **"프로젝트 설정" 클릭**
6. **"내 앱" 탭 클릭**
7. **Android 앱 섹션에서 "google-services.json" 버튼 클릭**
8. **파일 다운로드됨** ✅

---

### ✅ 2단계: 파일 복사하기

#### 🚀 방법 1: 자동 복사 (가장 쉬움!)

1. **파일 탐색기 열기**
   - Windows 키 누르고 "파일 탐색기" 입력 후 Enter
   
2. **이 폴더로 이동**:
   ```
   C:\Users\GetnShow - 37\Desktop\2026 gns app\scripts
   ```

3. **`copy_google_services.ps1` 파일 찾기**

4. **파일을 우클릭** → **"PowerShell로 실행"** 클릭

5. **자동으로 복사됨!** ✅

#### 📋 방법 2: 수동 복사

1. **파일 탐색기 열기** (Windows 키 + E)

2. **다운로드 폴더 열기**
   - 왼쪽에서 "다운로드" 클릭
   - 또는 주소창에 입력: `C:\Users\GetnShow - 37\Downloads`

3. **`google-services.json` 파일 찾기**

4. **파일을 복사**
   - 파일을 우클릭 → "복사" 클릭
   - 또는 파일 선택 후 Ctrl + C

5. **앱 폴더로 이동**
   - 주소창에 입력:
   ```
   C:\Users\GetnShow - 37\Desktop\2026 gns app\android\app
   ```

6. **파일 붙여넣기**
   - 빈 공간 우클릭 → "붙여넣기" 클릭
   - 또는 Ctrl + V

---

### ✅ 3단계: 확인하기

`android\app` 폴더에 `google-services.json` 파일이 있는지 확인하세요!

## ❓ 자주 묻는 질문

### Q: 파일을 찾을 수 없어요
**A**: 브라우저 하단의 다운로드 아이콘을 클릭하거나, 다운로드 폴더에서 "google-services.json" 검색

### Q: 폴더를 찾을 수 없어요
**A**: 파일 탐색기 주소창에 직접 입력:
```
C:\Users\GetnShow - 37\Desktop\2026 gns app\android\app
```

### Q: 파일이 이미 있어요
**A**: 덮어쓰기 해도 됩니다! (덮어쓰기 확인 시 "예" 클릭)

### Q: 복사가 안 돼요
**A**: 
- 파일이 열려있지 않은지 확인
- 관리자 권한이 필요한지 확인
- 다른 위치에 복사해보기
